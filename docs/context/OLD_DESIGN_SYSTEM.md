## 2. TIPOGRAFIA (Typography)

**Font Family**:
*   Headings: `Playfair Display` (Serif) - Authority, Tradition.
*   Body: `Inter` (Sans) - Readability, Modernity.

**Sizes**:
*   `text-base`: 18px (Boosted for seniors).
*   `text-lg`: 20px.
*   `text-xl`: 24px.

---

## 3. UI PATTERNS

### Cards (The "Paper" Look)
*   Background: White
*   Shadow: `shadow-md` (Soft diffuse gray)
*   Border: `1px solid slate-200`
*   No more "Glassmorphism" (Blur/Transparency). Use solid opacity.

### Buttons
*   **Primary CTA**: Blue 900 Background + White Text. Rounded Full.
*   **Urgency CTA**: Red 600 Background + White Text.
